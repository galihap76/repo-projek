1. Manakah singkatan dari HTML itu?


Jawaban pada kolom kesatu : Hyper Text Markup Language merupakan bahasa markah untuk bisa membuat sebuah website karena bisa di ibaratkan HTML merupakan tulang manusia tanpa adanya HTML sebuah halaman website tidak akan ada.
2. Siapakah pencipta javascript?


Jawaban pada kolom kesatu : beliau merupakan seorang programmer asal netscape dan javascript pertama kali diciptakan oleh pak brendan eich, seorang karyawan netscape, pada tahun 1995. netscape kala itu merupakan perusahaan software ternama yang dikenal dengan web browser miliknya, netscape navigator.
3. Bahasa pemrograman mana yang menggunakan berbasis client side?


Jawaban pada kolom kedua : karena javascript merupakan bahasa pemrograman tingkat tinggi yang berjalan kebanyakan dari web browser yang di jalankan juga pada interpreter. maksud dari client side ini bahasa pemrograman web yang pengolahan datanya dilakukan oleh komputer pengguna/pengunjung bagian dari browser nya tersendiri yang bisa kodenya di lihat juga oleh user maupun developer dan programmer.
4. Manakah yang termasuk database management system?


Jawaban pada kolom satu : MYSQL ini merupakan software database management system atau ini juga bisa di bilang server database. sederhana nya kegunaan MYSQL ini untuk menyimpan data seperti ketika website ada fitur login dan password. maka data akan tersimpan pada database management system MYSQL.
5. Manakah yang termasuk bahasa pemrograman untuk dekstop?


Jawaban pada kolom kedua : Delphi 7 ini merupakan lingkungan pengembangan dan berbasis windows. yang berorientasi bahasa pemrograman pascal karena delphi ini kebanyakan untuk membangun aplikasi dekstop namun bisa juga untuk membangun web dan perangkat mobile.
6. Dalam sebuah promise javascript ketika data berhasil di kirimkan keadaan apa yang di berikan ?


Jawaban pada kolom kesatu : dalam javascript ada sebuah fitur asynchronus yaitu promise dalam promise ketika data sudah terpenuhi dan berhasil sampai tujuan maka keadaan yang di berikan adalah fullfilled yang bisa di terjemahkan (terpenuhi) ketika seorang programmer bekerja menggunakan API (Application Programming Interface).
7. Kepanjangan ajax yaitu ?


Jawaban pada kolom satu : sederhana nya teknologi ini mempermudah ketika ada request client ke server jadi ketika ada perubahan data halaman pada website tidak perlu di refresh kembali.
8. Konsep IF pada bahasa pemrograman yaitu ?


Jawaban pada kolom ketiga : IF pada bahasa pemrograman ini agar ketika membuat sebuah program, program akan berpikir yang di berikan oleh programmer di mana program ini akan melakukan sesuatu sebagai contoh :
let namaku = 'galih';
if(namaku == 'galih'){
alert('Hai galih selamat datang'); }

Penjelasan nya tersebut apakah program di atas yang terletak pada di dalam if ('galih') tersebut sama dengan namaku yang di isi dengan 'galih'? jika sama tampilkan ('Hai galih selamat datang') jika tidak sesuai atau tidak sama, tidak di tampilkan pada program.
9. Konsep variabel pada pemrograman yaitu ?


Jawaban pada kolom kedua : sederhana nya variabel pada pemrograman untuk menyimpan sebuah nilai atau data program agar bisa di olah ataupun di gunakan nanti nya sesuai kebutuhan, pada variabel ini akan di simpan pada memori.
10. Konsep else pada pemrograman yaitu ?


Jawaban pada kolom satu : Konsep dan tugas else pada pemrograman ketika program memiliki sebuah percabangan seperti IF dan ELSE IF namun tidak ada yang benar maka di dalam kondisi else akan di jalankan.
